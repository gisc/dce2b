# Building a Second Brain with Agentic AI

Interactive e-learning course.

## How To Run

Open `index.html` in any modern web browser.

- No server, no internet, and no installation required.
- The optional "Narrate" buttons use your browser's built-in speech.

## Files

- `index.html`: The interactive course. Start here.
- `overview.html`: The course map with branching logic and Gagne event signposts. Open from the course's "Map" button or directly in a browser.

## Course Structure

- 6 decision points
- 3 learning activities
- 4 branching endings
- Structured on Gagne's Nine Events of Instruction

## Notes

- Keep both files in the same folder so the links work.
- Based on Andrej Karpathy's LLM Wiki pattern and Hermes Agent from Nous Research.
